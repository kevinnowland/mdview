# Markdown Site

Display nested folders of `.md` files as a static website with
support for mathjax (online only).

## Usage

```bash
mdview PATH/TO/MARKDOWN/FOLDERS
```

Set the `-d` flag to use darkmode.

For help:

```bash
mdview -h
```

## Installation

```bash
go install github.com/kevinnowland/mdview@latest
```
